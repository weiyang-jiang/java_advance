如果想要增肌同时减脂，必须力量结合有氧。力量训练如果怕动作不标准，可以先小重量做两组，然后动作熟悉后上大重量，不用做太多动作，但是每个动作尽量多做几组，注意是组，不是个，如果重量完不成一组就降点，动作标准，发力感明显，避免受伤最重要！其次新手最开始控制一点训练强度，给身体一个适应的时间，如果强度太大身体修复时间不足，过于疲劳人是无法坚持下去的，单次训练从开始热身到训练结束后拉伸不建议超过两个小时，因为超过两个小时后身体开始自我保护模式(这么说好理解点)，无论增肌减脂基本都没有什么效果了。另外还要做好训练计划，四练一休，五练一休，我是六练一休，都可以，分部位有针对性的训练效果最好，不要就知道几个动作，每个部位都练两下，回过头什么效果都没有。跑起来心率控制140以上，有氧运动跑步永远是王道，就是因为它能帮助人的心率长期控制在一个超级有效的燃脂心率范围中。

减脂的话需要注意没有局部减脂，虽然你肚子脂肪多，但是减脂也是全身性的。根据你体重不算太大来说，有氧开始可以用快走加上慢跑，心率要保证在130以上，时间最好在30-50分钟，身体适应后可以直接慢跑，后期逐步加速。

所以虽然开始跑起来有点难，但是坚持下来，相信我，你会爱上跑步的！最后健身的作息时间很重要，不要熬夜，会影响肌肉生成，多吃高蛋白食物，避免肌肉流失有助肌肉重组，减脂要少油少盐少糖，别吃零食，少吃水果和主食，主食可以用玉米红薯麦片等优质碳水替代，鸡肉，牛肉，瘦猪肉都可以吃，多吃青菜，火锅烤串啥的就暂时别碰了，吃一顿三天白练了。
